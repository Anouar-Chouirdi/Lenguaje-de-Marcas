Este archivo combina XML con HTML para mostrar información de libros sin conflictos de etiquetas.

Estructura del Archivo
- **Elemento raíz**: `<html>` con el espacio de nombres XHTML.
- **Secciones principales**:
  - `<head>`: Contiene el título del documento.
  - `<body>`: Contiene la reseña del libro dentro de `<div id="bookreview">`.
  - `<table>`: Muestra la información del libro en formato tabular.
